ขั้นตอนการติดตั้ง YOLOv5 บน Jetson Nano 

# อย่าลืมเชื่อมต่ออินเทอร์เน็ตกับบอร์ด jetson nano
-> เข้าไปที่ terminal แล้ว copy ไปวางทีละคำสั่งได้เลยครับ

1.==>อัพเดตระบบและติดตั้งไลบรารีพื้นฐาน

---------------------------------------------------------------------------------------------------------------------------
sudo apt update
---------------------------------------------------------------------------------------------------------------------------
sudo apt upgrade
---------------------------------------------------------------------------------------------------------------------------
sudo apt install git python3-pip python3-opencv
---------------------------------------------------------------------------------------------------------------------------

2.==>ติดตั้ง PyTorch และ TorchVision สำหรับ Jetson Nano

---------------------------------------------------------------------------------------------------------------------------
sudo apt install nvidia-jetpack
---------------------------------------------------------------------------------------------------------------------------
wget https://developer.download.nvidia.com/compute/redist/jp/v50/pytorch/torch-1.10.0-cp36-cp36m-linux_aarch64.whl
---------------------------------------------------------------------------------------------------------------------------
sudo pip3 install torch-1.10.0-cp36-cp36m-linux_aarch64.whil
---------------------------------------------------------------------------------------------------------------------------
sudo apt install python3-pip libjpeg-dev zlib1g-dev
---------------------------------------------------------------------------------------------------------------------------
git clone --branch v0.11.1 https://github.com/pytorch/vision torchvision
---------------------------------------------------------------------------------------------------------------------------
cd torchvision
---------------------------------------------------------------------------------------------------------------------------
sudo python3 setup.py install
---------------------------------------------------------------------------------------------------------------------------
cd ..
---------------------------------------------------------------------------------------------------------------------------

ตรวจสอบการติดตั้ง:

python3 -c "import torch; print(torch.__version__)"
python3 -c "import torchvision; print(torchvision.__version__)"

3. ติดตั้ง YOLOv5 และไลบรารีเสริม

---------------------------------------------------------------------------------------------------------------------------
git clone https://github.com/PrABpY/Fire-Alert-yolov5
---------------------------------------------------------------------------------------------------------------------------
cd Fire-Alert-yolov5
---------------------------------------------------------------------------------------------------------------------------
pip3 install -r requirements.txt
---------------------------------------------------------------------------------------------------------------------------

4. ทดสอบ YOLOv5 "รันดูว่ากล้องเปิดติดหรือไม่"

---------------------------------------------------------------------------------------------------------------------------
python Fire-detect.py
---------------------------------------------------------------------------------------------------------------------------
# วิธีปิด ให้กดตัว (Q) บนแป้นพิมพ์หลายๆครั้ง

5. ทดสอบ API 

---------------------------------------------------------------------------------------------------------------------------
python API.py
---------------------------------------------------------------------------------------------------------------------------
# วิธีปิด ให้กดตัว (Q) บนแป้นพิมพ์หลายๆครั้ง

เวลาจะรันใหม่อีกครั้ง ให้เข้าไปที่ Folder Fire-Alert-yolov5 แล้วรันคำสั่ง 

python3 API.py  

ใน terminal
